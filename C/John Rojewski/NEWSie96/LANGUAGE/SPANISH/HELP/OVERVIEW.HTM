Ayuda listado de artículos

Para cualquier grupo de noticias dado, la selección del mismo vía doble pulsación del botón izquierdo del ratón o la pulsación de la tecla [Return] en la ventana de Grupo de noticias suscrito iniciará la toma de la lista de artículos de noticias del servidor. Dicha lista es presentada en una ventana. Esta lista tiene el siguiente formato:

  • Número de artículo
  • Tema
  • Autor
  • Fecha de envío
  • Número de líneas del texto del mensaje


Se puede selecionar cualquiera de los artículos de esta lista, ya sea con el ratón o con las teclas de cursor del teclado. Cuando se ha escogido un artículo, éste es presentado en color rojo. En caso de usar una resolución monocromática, esta selección se indica con un subrayado.

Se reconocen la referencias de los artículos. Estas referencias son un grupo de artículos relacionados entre sí, y están marcados con el signo '+' para el primero de ellos, y con el signo '-' para el/los siguiente/s. Estas referencias se les da el formato dependiendo de cómo está configuradas las opciones de lectura de noticias Crear referencia y Cortar referencia. Si no esta seleccionada la opción Crear referencia, no se realiza el procesado de las referencias y los artículos se verán en orden numérico ascendente. Si en cambio esta opción está seleccionada, se verán tanto el primer artículo como los siguientes. Si está seleccionada la opción Cortar referencia, sólo se verá el primer artículo


Uso con el ratón:

  • Una doble pulsación del botón del ratón iniciará la recogida del artículo seleccionado del servidor de noticias.
  • Una sola pulsación selecciona el artículo (queda indicado con otro color).
  • Las funciones normales de las ventanas y la barra de desplazamiento.

Uso con el teclado:

  • La tecla [Return] iniciará la recogida del artículo seleccionado del servidor de noticias.
  • Las teclas de cursor para subir y bajar junto con la tecla [Shift] desplazan una página.
    • Las teclas de cursor para subir y bajar seleccionan (cambia de color) el artículo que esta encima o debajo del artículo escogido, y lo selecciona.
  • Las teclas de cursor para ir hacia la izquierda o hacia la derecha controlan la expansión/corte de las referencias individuales. Seleccionando el primer artículo y pulsando la tecla de cursor hacia la derecha presentará todos los artículos referenciados al primero. Pulsando la tecla de cursor para ir hacia la izquierda en el primer artículo o cualquiera de los siguientes, cortará la referencia justo donde está el primer artículo.
  • La tecla de cursor para ir hacia la derecha cortará una referencia.
  • La tecla de cursor para ir hacia la izquierda expandirá una referencia.
  • Pulsando la tecla [c] permitirá la cancelación de un anuncio previamente creado por usted. Esta orden será ignorada si el artículo no ha sido anunciado.
