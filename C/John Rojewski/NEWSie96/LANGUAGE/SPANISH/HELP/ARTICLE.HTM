Ventana de texto de los artículos

La ventana de texto de artículo presenta un artículo de una noticia. El artículo contiene una cabecera y el cuerpo del mensaje. La cabecera tiene las líneas: Autor, Tema, Fecha, etc. El cuerpo tiene el mensaje de respuesta introducido por el autor. Cada artículo leído esta disponible en su disco como NNNNNN.TXT, donde NNNNNN es el número del artículo. Estos fichero puede ser borrados con sólo seleccionarlos y pulsar la tecla [Delete] dentro de la ventana. De esta forma se proporciona un método para evitar la acumulación de mensajes no solicitados.


Uso con el ratón:

  • Las funciones normales de las ventanas y la barra de desplazamiento.
  • Una pulsación con el botón izquierdo del ratón en un enlace URL (subrayado y de color azul) iniciará un intento de recuperar y presentar en la ventana del navegador la página Web a la que apunta el enlace.

Uso con el teclado:

  • Las teclas del cursor para subir y bajar desplazan una línea de texto.
  • Las teclas de cursor para subir y bajar junto con la tecla [Shift] desplazan una página.
  • La tecla [Delete] borra de manera inmediata el artículo.

Si prefiere usar un programa como visor externo en vez de una ventana del propio NEWSie, puede indicar uno desde la caja de diálogo de Opciones de selección de directorios (Visualizador externo). Una vez hecho, seleccionar la opción Usar visualizador externo de la caja de diálogo de Opciones de lectura de las noticias, lo cual provocará que el NEWSie llame al programa externo pasándole como parámetro el nombre del fichero del artículo.
