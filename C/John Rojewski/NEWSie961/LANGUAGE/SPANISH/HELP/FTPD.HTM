Ficheros/Directorios por FTP

El propósito de esta ventana es la de permitir moverse a través de la estructura de directorios del servidor FTP, y bajar, subir, renombrar, o borrar ficheros.


Uso con el ratón:

  • Una doble pulsación del botón izquierdo del ratón recuperará la entrada seleccionada si esta es un fichero, o abrir un directorio si este es el caso.
  • Una sola pulsación se selecciona el fichero o el directorio (queda indicado con otro color).
  • Las funciones normales de las ventanas y la barra de desplazamiento.

Uso con el teclado:

  • La tecla [Return] recuperará la entrada seleccionada si esta es un fichero, o abrir un directorio si este es el caso.
  • Las teclas de cursor para subir y bajar junto con la tecla [Shift] desplazan una página.
  • Las teclas de cursor para subir y bajar seleccionan el fichero o el directorio que está encima o debajo del escogido, y lo selecciona.
  • La tecla [Insert] llama al selector de ficheros para escoger el fichero a subir al servidor en el directorio seleccionado.
  • La tecla [Shift] más la tecla [Insert] permitirá que el fichero sea salvado con un nombre diferente al que tiene originalmente en el servidor.
  • La tecla [Delete] borrará el fichero o el directorio seleccionado del servidor.
  • La tecla [r] permitirá que el fichero sea renombrado en el servidor.
  • La tecla [m] se emplea para crear un nuevo directorio en el servidor.
  • La tecla [Backspace] se emplea para volver al directorio anterior.
  • La tecla [a] pone la transferencia en modo ASCII (texto) para el siguiente fichero a recuperar o almacenar. Por defecto el modo es binario. Cada vez que se lea el contenido del directorio cambiará del modo ASCII al modo binario.
  • La tecla [b] pone la transferencia en modo BINARIO para el siguiente fichero a recuperar o almacenar. Por defecto el modo es binario.
  • La tecla [v] recupera el fichero seleccionado y lo presenta en una nueva ventana para ser leído. Simplifica la visualización de ficheros de texto índice e información.
