Servidor de FTP

El propósito de esta ventana es la de permitir la selección de un servidor de FTP para establecer una conexión, la edición o el borrado del mismo. Adicionalmente, se puede realizar la inserción de un nuevo servidor.


Uso con el ratón:

  • Una doble pulsación del botón del ratón se abre una conexión con el servidor FTP seleccionado.
  • Una sola pulsación se selecciona el servidor (queda indicado con otro color).
  • Las funciones normales de las ventanas y la barra de desplazamiento.

Uso con el teclado:

  • La tecla [Return] abrirá una conexión con el servidor FTP seleccionado.
  • Las teclas de cursor para subir y bajar desplazan una línea, y junto con la tecla [Shift] desplazan una página.
  • Las teclas de cursor para subir y bajar seleccionan el servidor que está encima o debajo del escogido, y lo selecciona.
  • La tecla [Insert] abre una caja de diálogo para añadir un nuevo servidor a la lista.
  • La tecla [Delete] borra de la lista el servidor seleccionado.
  • La tecla [?] abrirá a una caja de diálogo para editar los parámetros del servidor.
