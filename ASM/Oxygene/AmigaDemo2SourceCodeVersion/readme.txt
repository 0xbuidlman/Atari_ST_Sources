Amiga Demo 2 by Leonard/OXYGENE, Source code version

- You can build the demo by launching "build.bat" on windows plateforme
- The batch generate a MSA atari disk image file in the "Release" directory. You can run it using an emulator or write it on a real ATARI floppy disk.
- You can get details about the Leonard Demo System in "DemoSystem" directory
- 68k Source code is assembled on PC using as68, macro assembler written by Vincent Penne (Ziggy Stardust)
